# גננת בר - אתר למכירת חומרי חינוך

אתר מודרני ורספונסיבי למכירת קבצים וחומרי למידה לגננות ואמהות של ילדים בגילאי 2-6.

## תכונות האתר

### 🎨 עיצוב מודרני
- פלטת צבעים: סגול בהיר, קרם שמנת ולבן
- עיצוב רספונסיבי המותאם למובייל, טאבלט ומחשב
- אנימציות חלקות ואפקטים חזותיים
- תמיכה בכיוון RTL (עברית)

### 🧭 ניווט מתקדם
- תפריט קפוא בראש העמוד
- קישורי עוגן לגלילה חלקה בין סעיפים
- תפריט המבורגר למובייל
- הדגשת הסעיף הפעיל בתפריט

### 📱 אינטגרציה עם אינסטגרם
- הצגת פוסטים אחרונים מהחשבון
- קישור ישיר לפרופיל האינסטגרם
- עיצוב מותאם לפוסטים

### 🛒 מערכת מכירות מתקדמת
- גריד רספונסיבי של מוצרים
- קרוסלה של 3 תמונות לכל מוצר
- סינון מוצרים לפי קטגוריות
- חלון רכישה מתקדם
- מחירים ותיאורים מפורטים

### 📝 טופס יצירת קשר
- ולידציה מתקדמת של שדות
- הודעות שגיאה והצלחה
- עיצוב נגיש ונוח לשימוש

## מבנה הקבצים

```
website/
├── index.html              # עמוד ראשי
├── css/
│   ├── style.css          # עיצוב ראשי
│   └── responsive.css     # התאמות רספונסיביות
├── js/
│   ├── main.js           # פונקציות כלליות וניווט
│   ├── carousel.js       # קומפוננט קרוסלה
│   ├── instagram.js      # אינטגרציה עם אינסטגרם
│   └── products.js       # ניהול מוצרים ומכירות
├── images/               # תמונות האתר
└── assets/              # קבצים נוספים
```

## התקנה והפעלה

### דרישות מערכת
- דפדפן אינטרנט מודרני
- שרת ווב (למשיכה ממלאה)

### הפעלה מקומית
1. פתח את הקובץ `index.html` בדפדפן
2. או הרץ שרת HTTP פשוט:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve .
   ```

### הוספת תמונות
העתק תמונות לתיקיית `images/` עם השמות הבאים:

#### תמונות ראשיות:
- `hero-image.jpg` - תמונה ראשית
- `about-me.jpg` - תמונה אישית

#### תמונות מוצרים:
- `product-1-1.jpg`, `product-1-2.jpg`, `product-1-3.jpg`
- `product-2-1.jpg`, `product-2-2.jpg`, `product-2-3.jpg`
- וכו' (3 תמונות לכל מוצר)

#### תמונות אינסטגרם:
- `instagram-1.jpg` עד `instagram-6.jpg`

#### תמונות מאמרים:
- `article-1.jpg`, `article-2.jpg`

## התאמה אישית

### עדכון תוכן
ערוך את הקובץ `js/products.js` להוספת מוצרים חדשים:

```javascript
const newProduct = {
    id: 7,
    title: "שם המוצר",
    description: "תיאור המוצר",
    price: 30,
    images: [
        { src: "images/product-7-1.jpg", alt: "תיאור תמונה 1" },
        { src: "images/product-7-2.jpg", alt: "תיאור תמונה 2" },
        { src: "images/product-7-3.jpg", alt: "תיאור תמונה 3" }
    ],
    category: "קטגוריה",
    ageGroup: "2-4",
    downloadUrl: "#"
};
```

### שינוי צבעים
ערוך את המשתנים ב-`css/style.css`:

```css
:root {
    --primary-purple: #8B5FBF;    /* סגול ראשי */
    --light-purple: #B297D1;      /* סגול בהיר */
    --cream: #F5F1E8;             /* קרם */
    --soft-cream: #FBF9F4;        /* קרם רך */
}
```

### אינטגרציה עם אינסטגרם
להפעלת האינטגרציה האמיתית עם אינסטגרם, תצטרך:

1. **Instagram Basic Display API**
   - רישום באפליקציה Facebook Developers
   - קבלת Access Token
   - עדכון הקובץ `js/instagram.js`

2. **חלופה - שירות חיצוני**
   - שימוש בשירותים כמו Instagram Feed או Display Purposes
